# Linux file system commands

| Command  | Description                                                             | Example  |
| -------- | ----------------------------------------------------------------------- | -------- |
| `pwd`    | Print current working directory                                         | `pwd`    |
| `ls-l`   | List files and directories (long listing format)                        | `ls -l`  |
| `ls -la` | List files and directories (include hidden and use long listing format) | `ls -la` |
